## Requirement (verbatim — source of truth)

subtract two numbers and return the result

## Clarifications

(User answers to the auditor's questions, appended verbatim.)

- Inputs: two numbers, passed as arguments. Integers or decimals, negatives
  allowed.
- Operation: return the first number minus the second (e.g. 5 - 3 = 2, and
  order matters: 3 - 5 = -2).
- Output: the numeric result is returned (this is a plain function, no display
  styling required).
- Edge cases: assume valid numeric input; invalid/empty input is out of scope.
